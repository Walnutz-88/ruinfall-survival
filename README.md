# Ruinfall Survival

A lightweight 3D zombie-survival adventure prototype inspired by open-world survival games.

## Features
- Unique procedural world each launch (seeded terrain generation).
- Post-apocalyptic ruins scattered through the map.
- Zombie enemies that chase and damage the player.
- Survival stats (health and hunger).
- Non-block base-building with place/remove wall controls.

## Controls
- `WASD`: move
- `Mouse`: look around
- `Space`: jump
- `Shift`: sprint
- `Left Click`: place wall
- `R`: remove nearest wall

## Run on MacBook Pro
1. Install dependencies:
   - `npm install`
2. Start dev server:
   - `npm run dev`
3. Build production version:
   - `npm run build`
